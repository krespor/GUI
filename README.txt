Проект 'GUI' создан в Qt v5.14.1 с компилятором MinGW_64 и состоит из классов:
    1. mainwindow - класс граффического интерфейса. служит для обработки 
    действий пользователя в интерфейсе. при выборе допустимого изображения 
    (кнопка 'open image') активируется кнопка 'start', которая запускает 
    потоки класса сервера(server) и клиента(client).

    2. parser - класс для парсинга (!!)24-x разрядного изображения. На вход 
    конструктор принимает путь до файла, методы:
        QbyteArray getWH() - возвращает ширину и высоту изображения 
        пример: "32 32"

        QbyteArray getLine() - возвращает следующую строку по пиксклям 
        в формате r b g. пример строки из трёх пикселей (черный белый черный):
        "0 0 0 255 255 255 0 0 0"

    3. client - класс для передачи построчного изображения классу 
    сервера(server) по COM порту. запускается в отдельном потоке,
    создаёт объект класса parser, открывает порт, передаёт сначала ширину и 
    высоту(getWH()), затем построчно само изображение(getLine()) 
    и закрыват порт.

    4. server -  класс для принятия построчного изображения от класса
    клиента(client) по COM порту. запускается в отдельном потоке, 
    открывает порт, получает сначала ширину и высоту, затем построчно
    само изображение, сохраняя его в конструкции vector<vector<Pixel>>,
    закрыват порт и подаёт сигнал классу(mainwindow) для отрисовки.

*На данный момент поддерживаются только 24-х разрядные изображения. 
*Местами некорректное логирование процесса передачи данных.
*Можно передавать изображения меньше чем 32х32.
